
Getting Started
=================
NOTE: Currently, the project supports python 3.10.9: https://www.python.org/downloads/release/python-3109/


Install dependencies
::

    pip install -r requirements.txt
    


Project Setup
=================

Run the CDA exchange

::

    python ./run_market_client.py


Run the CDA client

::

    python ./run_market_client.py
