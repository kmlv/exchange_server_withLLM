Build container image with tag project:
docker build -t project .


Create subnet
docker network create --subnet=10.10.0.0/16 projectnet

Run instances

Run server
docker run -p 8082:8090 --net=projectnet --ip=10.10.0.2 --name=exchange project

Run a client
docker run -p 8083:8090 --net=projectnet --ip=10.10.0.3 -e SERVER_NAME='bingus' --name=alice project
